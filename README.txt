# IRF_Project

A program bevásárlólista összeállítására lesz alkalmas. Minden termék egy fő, termék osztály példánya lesz, nagyobb kategóriákra származtatott osztályokat fogok létrehozni (pl. tejtermékek), melyeknek egyedi tulajdonságaik lesznek (literenként lehet majd hozzáadni a listához). Bevásárlólistát lehet majd .csv-ből importálni és Excel fájlként exportálni.
